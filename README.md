# HomToChron

A trick-taking card-game, inspired by "Chronicle" by Seiji Kanai

## Run

    ./main.py

## Commands

#### Create a new game

    <player_name> new_game

#### Join or leave the game before it starts

    <player_name> join
    <player_name> leave

#### Begin the game

    <player_name> start

#### Play a card

Not implemented.
